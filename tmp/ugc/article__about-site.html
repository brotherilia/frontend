Об этом сайте

В процессе практики и родился этот сайт. Проект с этим дизайном был сделан давно, но применение нашел только сейчас. Основная его задача - формирование портфолио. Собственно с него (сайта), как одного из проектов, портфолио и начинается.

Это уже третья версия верстки сайта. В ней запланирована реализация адаптивной "резиновой" сетки и стилизация для 4-х вьюпортов. На данный момент работает только мобильная "версия".

Степень реализации: полная
Адаптивность: mobile (в разработке: tablet, desktop, desktop wide)
Сетка: responsive
Методология: БЭМ (именование классов)
CSS препроцессор: LESS
Автоматизация: Grunt
Исходные тексты: source, build
