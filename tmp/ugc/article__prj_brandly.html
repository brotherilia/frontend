Brandly

Проект на доработке

Учебная верстка из рассылки HTML Academy. Одна страница. Адаптивная "резиновая" верстка для 2-х вьюпортов, адаптивная графика.

Степень реализации: верстка
Адаптивность: mobile, desktop
Сетка: responsive
Методология: БЭМ (именование классов)
CSS препроцессор: LESS
Автоматизация: Grunt
